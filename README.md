# AI Image Recognition System

Deep learning image classification system with multi-class recognition, transfer learning, and REST API for production deployment.

## Features

- Multi-class image classification with transfer learning
- Pre-trained model support (ResNet, VGG, EfficientNet)
- REST API endpoint for inference
- Batch processing for large image datasets
- Confidence scoring and top-K predictions
- Model evaluation metrics and confusion matrix

## Tech Stack

Python, TensorFlow, Keras, Flask, NumPy, Pillow

## Getting Started

### Prerequisites

- Python 3.8+
- pip

### Installation

```bash
git clone https://github.com/karthik-idikuda/AI-Image-Recognition-System.git
cd AI-Image-Recognition-System
pip install -r requirements.txt
```

### Usage

```bash
python predict.py --image path/to/image.jpg
```
